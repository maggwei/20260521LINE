<h3>公廁與垃圾桶查詢機器人</h3><br>
這是一個可以幫你快速找到附近「公廁」與「垃圾桶」的 LINE 機器人。<br>

<br>
機器人好友ID：@887tmbub<br>
<br>
連結的公開資料來源：<br>
https://data.gov.tw/dataset/30794<br>
https://data.gov.tw/dataset/121355<br>
<br>
使用說明：<br>
查詢方式：在 LINE 對話框點選「尋找附近公廁」、「尋找台北市垃圾桶」，並傳送你現在的位置。<br>
查詢結果：機器人會立刻回傳離你最近的三個公廁或垃圾桶資訊。<br>
回主選單：輸入「選單」。<br>
